# **App Name**: EasyTest

## Core Features:

- Test Import: Teachers can upload a Word document containing an ABCD test with images and correct answers directly from their computer.
- Difficulty Scaling: Teachers can adjust difficulty sliders to set the grade scale for 1-5.
- Link Generation: Generate a unique test link for distribution to student groups via school Google email accounts.
- Student Authentication: Students enter their name, surname, and class before taking the test.
- Test Execution: Students can complete the test on any PC or mobile device within the app interface, without needing to install or configure anything beyond accessing the URL.
- Result Display: Students receive an immediate grade and a review of correct/incorrect answers.
- Grade Aggregation: Teachers can view student results for each test, broken down by student.
- Results Export: Teachers can export test results into a CSV file for further analysis.

## Style Guidelines:

- Primary color: A friendly light blue (#A0D2EB), offering a sense of calm and trustworthiness.
- Background color: Very light blue (#EBF4FA), close in hue to the primary color, but much less saturated.
- Accent color: A light, warm, analogous yellow-orange (#F4D0A4) to draw attention to calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif with a balance of modern appeal and approachability, appropriate for both headlines and body text.
- Use simple, clear icons to represent actions and test elements.
- Maintain a clean, uncluttered layout to ensure ease of navigation and focus.
- Subtle animations and transitions to provide feedback and engagement during test-taking.
